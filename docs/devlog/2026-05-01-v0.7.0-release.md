# 2026-05-01 ‚Äî v0.7.0 release

## What shipped

Single feature release: G9 dashboard redesign GA.

**Features (G9 dashboard):**
- 3-level analytics surface (engineer ¬∑ project ¬∑ org) with bookmarkable URL state (`?role=&id=&period=&compare=`)
- CWD-aware landing ‚Äî git remote ‚Üí project key resolution
- Per-level hero tiles (DORA composite, attribution, autonomy, retention, $/task, etc.)
- DORA scorecard (org + project) ‚Äî Elite/High/Medium/Low ratings, stale-banner >24h
- Attribution composite tile + 28d sparkline
- Intent feed (layer-4 events) ‚Äî engineer + project filters, click-to-expand
- Insight engine (`internal/insights/`) ‚Äî 5 SQL heuristics rendered as cards with drilldown URLs
- Run-row inline expand (iterations + intent) + engineer drilldown (50 runs, retention sparkline)
- Mobile responsive @375 (no horizontal overflow)

**Cutover:**
- `--experimental` flag fully removed; G9 is the only dashboard
- Legacy `dashboardHTML` const + `newExperimentalDashboardMux` deleted (~1300 lines)
- Legacy panels (Overview, Agents, Cost charts, Recent Runs, Quality KPI) untouched

**Fixes (post-cutover, same release):**
- DORA scorecard now scopes by project (was always returning org snapshot)
- Project view intent feed now renders (was hidden by `applyRoleVisibility`)
